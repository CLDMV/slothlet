# Slothlet Bugs

This document tracks currently known bugs in Slothlet v3. All bugs documented in the v2 release ([`docs/v2/root/BUGS.md`](docs/v2/root/BUGS.md)) were resolved during v3 development and are not reproduced in the current architecture.

---

## Reporting a Bug

Before filing a bug report:

1. Check this document for already-known issues
2. Search existing [GitHub Issues](https://github.com/CLDMV/slothlet/issues)
3. Reproduce with a minimal example
4. Include Node.js version, Slothlet version, and mode (`eager` / `lazy`)

To report: open a [GitHub Issue](https://github.com/CLDMV/slothlet/issues/new) with a clear description, reproduction steps, and expected vs actual behavior.

For **security vulnerabilities**, see [`SECURITY.md`](SECURITY.md) - do not open a public issue.
